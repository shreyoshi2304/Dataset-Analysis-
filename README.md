# Dataset Analysis and Applied Machine Learning

## Overview

This repository contains a collection of projects developed as part of a Data Science, Machine Learning, and AI course, along with additional independent exploration.

The work focuses on analyzing real-world datasets (such as YouTube and Facebook data) and applying computational methods to extract patterns, build models, and generate insights.

## Scope of Work

* Exploratory data analysis on social media datasets
* Statistical and visual analysis of user behavior and trends
* Implementation of regression and basic machine learning models
* Development of simple chatbot systems using transformer-based models (Hugging Face)

## Datasets

* YouTube dataset (engagement, views, trends)
* Facebook dataset (interaction patterns, user activity)

## Methodology

* Data cleaning and preprocessing
* Feature exploration and visualization
* Model fitting and evaluation
* Application of pre-trained NLP models for chatbot development

## Key Insights

* Identification of trends in user engagement across platforms
* Understanding relationships between content features and performance
* Exposure to real-world variability and noise in data

## Chatbot Component

As part of the project, I implemented basic chatbot systems using models from the Hugging Face ecosystem, exploring:

* Text generation
* Conversational response behavior
* Practical limitations of pre-trained models

## Tech Stack

* Python
* NumPy, Pandas
* Matplotlib
* Scikit-learn
* Hugging Face Transformers


## Future Work

* Apply more advanced machine learning models
* Perform time-series analysis on engagement data
* Improve chatbot coherence and contextual understanding
* Extend analysis to larger and more diverse datasets

## Note

While this work originated from coursework, several components were extended independently to explore practical applications of machine learning and data analysis.
